龚俊衡

当前公司

现居地

中国, 四川, 成都

简介

多年大数据和机器学习平台架构相关经验, 熟悉数据仓库, 推荐系统, 用户流失预测 以及 群组分析.

熟悉常用大数据技术栈, 比如 Hadoop, Hbase, (较老版本), Elasticsearch, Spark 和 常用 AWS 组件, 例如 Kinesis, DynamoDB, EMR, S3 等

熟悉常用机器学习算法, 具备阅读和复现相关研究文献能力.

作为团队领袖带领团队在大数据计算/机器学习平台以及机器学习落地相关领域有充分的经验.

作为工程师, 熟悉 Python, Scala, Java 和大多数机器学习工具箱, 并对 Akka, Spark, Lagom 有较为深入的了解, 同时对于 K8S 有实际应用经验.

作为分享者, 多次被大数据和机器学习相关讲座和会议邀请, 包括 AWS Summit 2017.

终生学习者.

教育背景

成都信息工程学院 (2003 - 2006) — 大专, 软件工程

学生会, 软件协会分会长

经历

成都品果科技有限公司 - 大数据 / 机器学习 架构师

从 2016.6 到 现在 位于 中国, 四川, 成都

组建团队以 AWS 为基础设施, 基于 Spark 和 Tensorflow构建大数据和深度机器学习平台. 被应用于大量公司产品, 包括 Camera360, Mix, One Camera, Salad Video 等, 服务于数亿用户. 在数据治理, 业务分析, 内容推荐和许多其他AI 相关业务领域做出贡献.

Remark Holdings - 大数据架构师

从 2015.1 到 2016.6 位于 中国, 四川, 成都

作为大数据架构师, 带领团队构建基于 Hadoop 和 HBASE (CDH4) 的数据湖产品, 并负责公司的分布式爬虫系统(基于 Scala 和 Python), 并基于 Spark 构建机器学习服务.

Symbio - 架构师

从 2012.10 到 2015.1 位于 中国, 四川, 成都

作为架构师和高级开发人员, 参与了大量大数据和机器学习项目, 包括电子商务, 车联网和数据分析相关项目,

ThoughtWorks - 高级软件工程师

从 2011.10 到 2012.6 位于 中国, 四川, 成都

作为高级开发工程师, 熟悉敏捷开发流程, 参与了 Sun Crop. 项目, 作为 Java 开发工程师, 帮助以服务总线架构重构客户遗留系统.

中软国际 - 模块设计工程师

从 2010.9 到 2011.10 位于 中国, 成都 和 杭州

外包到华为作为高级软件工程师, 并被客户任命为模块设计工程师, 参与 LTE 相关项目, 比如 OM 和 基站模拟系统.

Zadmobile - 软件工程师

从 2006.9 到 2010.6 位于 中国, 上海
