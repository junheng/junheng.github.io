GIT 提交时的特殊字符

1月 11 2015 几秒 read (About 86 words)

Intellij IDEA 偶尔在提交代码时候会出错, 产生包含特殊字符的路径和文件, 普通的办法无法删除 需要利用 -o -z 参数才能够正常添加到改动:

git ls-files -d -m -o -z -x | xargs -0 git update-index –add –remove

-o -z 包含特殊字符

-x 用于应用忽略列表
